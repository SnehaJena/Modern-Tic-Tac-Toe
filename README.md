# 🎮 Modern Tic-Tac-Toe

A modern and responsive Tic-Tac-Toe game built using **HTML, CSS, and JavaScript**. The game features a clean user interface, smooth interactions, winner detection, draw detection, and score tracking.

## ✨ Features

- 🎯 Two-player gameplay
- 🏆 Winner detection
- 🤝 Draw detection
- 📊 Live scoreboard
- 🔄 Restart game
- 🆕 New Game
- 📱 Responsive design
- ✨ Smooth hover animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
Modern-Tic-Tac-Toe/
│── index.html
│── style.css
│── script.js
│── README.md
```

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/SnehaJena/Modern-Tic-Tac-Toe.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
